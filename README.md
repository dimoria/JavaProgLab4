### Дії з текстом  
> У кожному реченні заданого тексту поміняти місцями перше та останнє слово,  
> не змінюючи довжини речення.

## Структура програми  
Було створено такі класи: 
### **1. Letter**
- зберігає один символ
- використовується для побудови слова

### **2. Word**
- складається з масиву об’єктів `Letter`
- містить методи роботи зі словом

### **3. PunctuationMark**
- зберігає розділовий знак, що завершує речення

### **4. Sentence**
- складається з масиву `Word`
- має кінцевий розділовий знак (`PunctuationMark`)
- реалізує метод `swapFirstAndLastWord()`

### **5. Text**
- складається з масиву `Sentence`
- нормалізує пробіли (замінює багато пробілів/табів на один)

### **6. TextProcessor**
- головний клас з методом `main()`
- виконує дію з варіанта №4, використовуючи створені класи
